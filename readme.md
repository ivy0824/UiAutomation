## install
1. git clone git@git.blacklake.tech:testing/automation.git
2. cd automation
3. npm install
4. npm run test

## getting started
- chai(assert): http://chaijs.com/api/assert/
- mocha: https://mochajs.org/
- puppeteer: https://github.com/GoogleChrome/puppeteer
